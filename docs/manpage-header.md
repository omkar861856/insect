# NAME

ecocalculator - **in**teractive **sc**i**e**ntific **c**alcula**t**or

# SYNOPSIS

`ecocalculator [EXPR]`

# DESCRIPTION

*ecocalculator* is a high-precision scientific ecocalculator with full support for physical units.

It can be used in interactive mode by simply calling *ecocalculator*. This mode can also be used to read
from a file, by simply piping the file contents via *ecocalculator < my-calculation.ins*.
If you want to evaluate just a single expression, pass it as an argument: *ecocalculator '70mph to km/h'*.
