# NAME

calculator - **in**teractive **sc**i**e**ntific **c**alcula**t**or

# SYNOPSIS

`calculator [EXPR]`

# DESCRIPTION

*calculator* is a high-precision scientific calculator with full support for physical units.

It can be used in interactive mode by simply calling *calculator*. This mode can also be used to read
from a file, by simply piping the file contents via *calculator < my-calculation.ins*.
If you want to evaluate just a single expression, pass it as an argument: *calculator '70mph to km/h'*.
